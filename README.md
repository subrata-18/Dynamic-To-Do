
# Dynamic To-Do 📋⚡

A clean and simple **productivity mobile app** built with **React Native and Expo** to help users manage tasks efficiently.
This is my **first mobile app project**, focused on learning mobile development and building a practical productivity tool.

---

## 🚀 Features

* ✅ Create and manage tasks easily
* 📝 Add tasks dynamically
* ✏️ Edit existing tasks
* 🗑️ Delete tasks
* 📱 Clean and minimal mobile UI
* 💾 Local task storage for persistence
* ⚡ Fast and responsive performance

---

## 🛠️ Built With

* **React Native**
* **Expo**
* **TypeScript (TSX)**
* **AsyncStorage** for saving tasks locally

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/dynamic-todo-app.git
```

Navigate into the project:

```bash
cd dynamic-todo-app
```

Install dependencies:

```bash
npm install
```

Run the Expo development server:

```bash
npx expo start
```

You can run the app on:

* Android Emulator
* iOS Simulator
* Expo Go on your phone
